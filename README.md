## How to View Locally
1. Clone the repository:  
   ```bash
   git clone https://github.com/OlganeOlga/Comportfolio.git
   ```     
2. Open index.html in a browser

3. SCSS files can be compiled to CSS using Sass:
```sass --watch scss/main.scss:css/main.css```
